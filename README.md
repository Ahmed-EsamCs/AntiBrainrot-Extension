# 🧠 AntiBrainrot

**Stay Locked In.**

AntiBrainrot is a Chrome Extension designed to help you stay focused during study or work sessions by minimizing distractions and encouraging deep work.

Unlike traditional website blockers, AntiBrainrot is built around the concept of **Focus Missions**. Instead of simply blocking websites, it creates a distraction-free environment where you can work toward a specific goal while tracking your productivity.

---

## ✨ Features

### 🎯 Focus Sessions

* Create focused work sessions.
* Set a custom duration.
* Start, pause, resume, and end sessions.
* Real-time countdown timer.

### 🌐 Smart Website Blocking

* Allow only productive websites during a focus session.
* Block distracting websites automatically.
* Customizable allowlist and blocklist.

### 📝 Task Management

* Create and organize tasks.
* Assign focus sessions to tasks.
* Track time spent on each task.
* Mark tasks as completed.

### 📊 Productivity Analytics

* Track total focus time.
* View completed sessions.
* Monitor daily productivity.
* Analyze your work habits over time.

### 🧠 Brain Evolution

The extension icon evolves as you stay focused.

The more productive time you accumulate, the brighter and more energetic your brain becomes, representing your progress and discipline.

### 💬 Motivational Experience

Instead of displaying a plain blocked page, AntiBrainrot motivates you to stay focused with encouraging messages and a clean, distraction-free interface.
